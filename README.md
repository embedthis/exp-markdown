exp-markdown
===

Expansive plugin for Markdown files.

Provides the 'markdown' service.

### To install:

    pak install exp-markdown

### To configure in expansive.json:

* enable &mdash; Enable the compile-markdown-html service to process Markdown files.

```
{
    services: {
        'markdown': {
            enable: true,
        }
    }
}

```
